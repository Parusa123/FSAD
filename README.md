# 🚀 Full Stack Application Development Lab

## 📌 Overview
This repository contains all the work, experiments, and projects developed as part of the **Full Stack Application Development Lab**. The lab focuses on building complete web applications using modern frontend and backend technologies, along with database integration.

---

## 🎯 Objectives
- Understand the fundamentals of full stack development  
- Build responsive and dynamic web applications  
- Integrate frontend with backend services  
- Work with databases for data storage and retrieval  
- Gain hands-on experience with real-world project development  

---

## 🛠️ Technologies Used

### Frontend
- HTML5  
- CSS3  
- JavaScript  
- React / Bootstrap (optional)

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB / MySQL  

### Tools
- Git & GitHub  
- VS Code  
- Postman
